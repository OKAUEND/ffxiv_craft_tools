# FFXIV CRAFT Tools
This tool assists materials management in production activities within **FFXIV**.

##Build Setup
>## install dependencies
>```
>yarn install
>```
>### Compiles and hot-reloads for development
>```
>yarn run serve
>```
>### Compiles and minifies for production
>```
>yarn run build
>```
>### Run your unit tests
>```
>yarn run test:unit
>```
